# 🛒 ShopSphere

A modern and responsive e-commerce web application built using **React** and **Vite**.

## ✨ Features

- Product Listing
- Product Details
- Search Products
- Category Filter
- Shopping Cart
- Wishlist
- User Login & Signup
- Checkout
- Order Success Page
- Responsive Design
- Toast Notifications
- Form Validation
- Smooth Animations

## 🛠️ Technologies Used

- React
- Vite
- React Router DOM
- Context API
- React Icons
- React Toastify
- CSS3
- Local Storage
